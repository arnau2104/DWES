Configuració de l'Entorn Local amb XAMPP

Crear el Projecte en XAMPP

Navega a la carpeta C:\xampp\htdocs.

Crea un directori per al projecte. En aquest cas, es crea “student068”.

Clonar el Repositori

Clona el repositori de codi al directori del projecte:

git clone git@github.com:erikmozos/student068.git C:\xampp\htdocs\student068

Instal·lar Tailwind CSS amb CDN

Per evitar la configuració de Tailwind amb npm, podem instal·lar-lo de manera més senzilla utilitzant el CDN.

1. Obre el fitxer index.html (o qualsevol fitxer HTML que estiguis utilitzant per al teu projecte).

2. Afegeix el següent en l’etiqueta <head> del teu arxiu HTML per carregar Tailwind CSS a través del CDN:

<link href="https://cdn.jsdelivr.net/npm/tailwindcss@3.0.24/dist/tailwind.min.css" rel="stylesheet">

Aquesta línia carregarà l’última versió de Tailwind CSS directament des del CDN. No cal cap configuració addicional per instal·lar Tailwind ni compilar els fitxers CSS.

Iniciar el Projecte

1. Després d’haver clonado el projecte i inclòs el CDN de Tailwind CSS, pots començar a treballar amb el teu projecte.

2. Per veure el teu projecte en acció, simplement obre el fitxer index.html en el teu navegador o accedeix a través de XAMPP:

Obre el navegador i accedeix a: http://localhost/student068/

Pujar Arxius al Servidor amb FileZilla

1. Configurar FileZilla:

  • Obre FileZilla i accedeix al Gestor de Llocs.
  • Configura un nou lloc amb els següents detalls:
    • Host: Direcció del servidor.
    • Protocol: FTP.
    • Tipus d’accés: Normal.
    • Usuari: El teu nom d’usuari FTP.
    • Contrasenya: La teva contrasenya FTP.

2. Transferir Arxius:

  • Navega a la carpeta del teu projecte local (per exemple, C:\xampp\htdocs\student068).
  • Navega a la carpeta de destí al servidor.
  • Arrossega els arxius des de la finestra esquerra a la dreta per transferir-los al servidor.
