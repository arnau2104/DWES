Configuració de l'Entorn Local amb XAMPP

Crear el Projecte en XAMPP

Crea un directori per al projecte en la carpeta C:\xampp\htdocs. En aquest cas, es crea “student067”.

Instal·lar Tailwind CSS amb CDN :

Per evitar la configuració de Tailwind amb npm, podem instal·lar-lo de manera més senzilla utilitzant el CDN.

1. Obre el fitxer index.html (o qualsevol fitxer HTML que estiguis utilitzant per al teu projecte).

2. Afegeix el següent en l’etiqueta <head> del teu arxiu HTML per carregar Tailwind CSS a través del CDN:

<link href="https://cdn.jsdelivr.net/npm/tailwindcss@3.0.24/dist/tailwind.min.css" rel="stylesheet">

Aquesta línia carregarà l’última versió de Tailwind CSS directament des del CDN. No cal cap configuració addicional per instal·lar Tailwind ni compilar els fitxers CSS.

Iniciar el Projecte

Per iniciar el projecte en local hem d0iniciar primer el servidor xampp

Obre el navegador i accedeix a: http://localhost/student067/

Tambe el pots veure a https://remotehost.es/student067/dwes/

Pujar Arxius al Servidor

Per pujar archius al servidor utilitzarem la aplicacio de FileZilla

1. Configurar FileZilla:

  • Obre FileZilla i ompleix els següents camps:
    • Host: Direcció del servidor.
    • Protocol: FTP.
    • Tipus d’accés: Normal.
    • Usuari: El teu nom d’usuari FTP.
    • Contrasenya: La teva contrasenya FTP.

2. Transferencia de Fitxers Arxius:

Una vegada establerta a la connexio arrastra els fitxers que vols oujar el servidor i ja ho tindriem!
